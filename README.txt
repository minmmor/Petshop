DASHBOARD PETSHOP - FRONT-END ESTÁTICO

Arquivos:
- index.html
- style.css
- script.js

Como abrir:
1. Extraia a pasta.
2. Abra o arquivo index.html no navegador.

O projeto contém:
- Dashboard geral
- Clientes
- Pets
- Agendamentos
- Serviços
- Funcionários
- Produtos
- Fornecedores
- Vendas
- Compras
- Relatórios baseados nas views

Observação:
Os formulários e botões são demonstrativos. Para salvar no PostgreSQL será necessário criar uma API/backend, por exemplo com Node.js, PHP, Python Flask ou Java.
