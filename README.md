# 🐾 Dashboard Petshop

Este projeto consiste em uma interface de gerenciamento (Dashboard) para um Petshop, integrada conceitualmente a um modelo de banco de dados utilizando PostgreSql. A interface foi desenvolvida para ilustrar como os dados de funcionamento do negócio são visualizados e manipulados.

## 🚀 Como Abrir o Projeto

Para visualizar a interface do Dashboard no seu computador, siga os passos abaixo:

1. **Extraia a pasta** compactada contendo os arquivos do projeto (caso esteja em formato `.zip` ou `.rar`).
2. Navegue até a pasta extraída.
3. Dê um duplo clique no arquivo **`index.html`** para abri-lo diretamente no seu navegador de preferência.

---

## 📂 Estrutura de Arquivos Front-end

O sistema visual é composto por três arquivos principais:
* 📄 **`index.html`**: Estrutura robusta contendo todas as telas, seções e formulários do sistema.
* 🎨 **`style.css`**: Identidade visual e estilização moderna, responsiva e intuitiva.
* ⚡ **`script.js`**: Lógica de alternância entre as telas do menu e comportamento dinâmico dos elementos demonstrativos.

---

## 🖥️ O que o Projeto Contém

A interface simula um sistema ERP completo de Petshop, dividido nas seguintes seções:

* **📊 Dashboard Geral**: Painel com métricas rápidas e visão unificada do negócio.
* **👥 Clientes**: Cadastro e listagem dos tutores dos animais.
* **🐕 Pets**: Registro dos animais vinculados aos seus respectivos donos.
* **📅 Agendamentos**: Controle de horários para banho, tosa e consultas.
* **💉 Serviços**: Tabela de serviços oferecidos com seus respectivos valores.
* **👔 Funcionários**: Gestão da equipe (veterinários, tosadores, atendentes).
* **📦 Produtos**: Controle de estoque de rações, brinquedos e acessórios.
* **🚚 Fornecedores**: Registro dos parceiros comerciais de suprimentos.
* **💰 Vendas**: Frente de caixa para registrar a saída de produtos e serviços.
* **🛒 Compras**: Registro de entrada de mercadorias e reposição de estoque.
* **📈 Relatórios**: Telas analíticas alimentadas diretamente por **Views lógicas do Banco de Dados**.
